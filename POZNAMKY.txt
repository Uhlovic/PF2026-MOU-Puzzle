PF 2026 - NOVOROČNÍ PUZZLE - POZNÁMKY K PROJEKTU
===============================================

DATUM: 2025-11-19
STAV: HOTOVO ✅

POPIS PROJEKTU:
---------------
Interaktivní puzzle aplikace pro novoroční přání 2026
Centra onkologické prevence MOU. Puzzle zobrazuje
vizualizaci nového pavilonu. Minimalistický design
s glassmorphism efekty.

SOUBORY V PROJEKTU:
-------------------
- index.html                           (HTML struktura)
- style.css                            (CSS styling + animace)
- script.js                            (JavaScript logika)
- 2021_12_09_BK_MOU_CP_25 (1).jpg     (hlavní obrázek puzzle)
- BG_new.png                           (nové pozadí)
- logo.svg                             (logo MOÚ)
- Matter-Regular (1).otf               (firemní font - regular)
- Matter-SemiBold (2).otf              (firemní font - semibold)

HLAVNÍ FUNKCE:
--------------
✅ Puzzle 4x3 (12 dílků) - rozměry 240x180px (desktop)
✅ Progressive reveal - zobrazí se 3 náhodné dílky na začátku
✅ Po správném umístění se objeví další dílek
✅ Drag & Drop funkcionalita
✅ Vizuální feedback:
   - Správně: oranžový rámeček (#f04600)
   - Špatně: červený rámeček s shake animací
✅ Tlačítka v pravém horním rohu:
   - Reset (Začít znovu) - oranžové
   - Nápověda - oranžové s bílou ikonou
✅ Nápověda overlay s instrukcemi
✅ Statické pozadí (BG_new.png)
✅ Merge animace po dokončení (3s delay)
✅ Completion overlay s novým textem a odkazem
✅ Logo + text v pravém dolním rohu
✅ Firemní fonty Matter (Regular + SemiBold)
✅ Plně responzivní (desktop, tablet, mobile)
✅ Minimalistický glassmorphism design

BAREVNÉ SCHÉMA:
---------------
- Hlavní barva: #f04600 (oranžová)
- Gradienty: #f04600 → #ff6b35
- Bílé akcentní prvky
- Minimalistické průhledné boxy: rgba(0, 0, 0, 0.25) + blur(10px)
- Bílý text na průhledných boxech

VELIKOSTI PUZZLE:
-----------------
Desktop (>1200px):   240x180px dílky
Tablet (768-1200px): 180x135px dílky
Mobile (<768px):     90x67.5px dílky

TECHNOLOGIE:
------------
- Vanilla JavaScript (ES6+)
- HTML5 + CSS3
- Drag & Drop API
- CSS Grid layout
- CSS Animations (shake, slideIn, pulse, firework)
- @font-face pro vlastní fonty
- Media queries pro responzivitu
